# Tjenare Logistics Status

This repository monitors the uptime and response times of [Tjenare Logistics](https://logistics.tjenaremannen.se) using [Upptime](https://upptime.js.org).

## Monitored Endpoints

- **Logistics Platform** — `https://logistics.tjenaremannen.se`
- **API Health** — `https://logistics.tjenaremannen.se/api/health`
